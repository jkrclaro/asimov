# Podplayer

![Vercel](https://therealsujitk-vercel-badge.vercel.app/?app=podplayer)

Podcast player using React.js

## Installation

```sh-session
npm install
```

## Usage

Run the app locally

```sh-session
npm run start
```

Lastly, you will see the URL can be visited in

```
localhost:3000
```

Use this URL in your browser to see the app.
